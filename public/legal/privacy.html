Politique de confidentialité

Dernière mise à jour : 1er janvier 2025

🔒 ShortFactory ne vend jamais vos données personnelles à des tiers et ne les utilise qu'aux seules fins décrites dans cette politique.

1. Responsable du traitement

ShortFactory est le responsable du traitement de vos données personnelles dans le cadre de l'utilisation de l'Application.

2. Données collectées

Nous collectons uniquement les données strictement nécessaires au fonctionnement de l'Application :

  • Données de compte TikTok : identifiant unique (open_id), nom d'affichage, URL de l'avatar — obtenus via l'API TikTok après votre autorisation explicite
  • Tokens d'accès OAuth : access token et refresh token TikTok, stockés localement sur le serveur pour permettre la publication en votre nom
  • Fichiers vidéo temporaires : les vidéos traitées sont stockées temporairement sur le serveur (dossiers tmp/ et output/) et automatiquement supprimées après traitement
  • Journaux techniques : logs d'erreurs et d'activité à des fins de débogage, sans données personnelles identifiantes

3. Finalités du traitement

Vos données sont utilisées exclusivement pour :

  • Authentifier votre compte TikTok via OAuth 2.0
  • Afficher vos informations de compte dans l'interface de l'Application
  • Publier des vidéos sur votre compte TikTok en votre nom, selon vos instructions
  • Maintenir votre session active via le renouvellement automatique des tokens

4. Données TikTok — scopes utilisés

L'Application demande les autorisations (scopes) TikTok suivantes :

  • user.info.basic : lecture de votre nom d'affichage et de votre avatar, uniquement pour les afficher dans l'interface
  • video.upload : téléversement de fichiers vidéo sur les serveurs TikTok
  • video.publish : publication des vidéos sur votre profil TikTok

Nous n'accédons à aucune autre donnée de votre compte TikTok (vidéos existantes, abonnés, messages, etc.).

5. Conservation des données

  • Tokens OAuth : conservés jusqu'à leur révocation par vos soins ou jusqu'à leur expiration naturelle
  • Fichiers vidéo temporaires : supprimés automatiquement après encodage et publication (généralement quelques minutes à quelques heures)
  • Données de profil : conservées tant que votre compte est connecté à l'Application

6. Partage des données

Nous ne partageons vos données avec aucun tiers, à l'exception des APIs officielles de TikTok nécessaires au fonctionnement du service. Nous n'utilisons aucun outil d'analytics, de publicité ou de tracking tiers.

7. Sécurité

Les tokens OAuth sont stockés dans un fichier JSON sur le serveur, protégé par les mécanismes de sécurité du système d'exploitation. Les communications avec les APIs TikTok s'effectuent exclusivement via HTTPS.

8. Vos droits

Vous disposez des droits suivants concernant vos données :

  • Droit d'accès et de rectification : vous pouvez consulter les données vous concernant
  • Droit à l'effacement : vous pouvez supprimer votre compte connecté à tout moment depuis l'interface
  • Droit de révocation : vous pouvez révoquer les autorisations TikTok depuis les paramètres de votre compte TikTok → Applications connectées

9. Cookies

L'Application n'utilise pas de cookies de traçage ou publicitaires. Seul le stockage de session du navigateur (sessionStorage) est utilisé pour maintenir l'état de l'interface.

10. Modifications

Nous nous réservons le droit de modifier cette politique à tout moment. Toute modification sera publiée sur cette page avec une date de mise à jour.

11. Contact

Pour exercer vos droits ou pour toute question relative à la présente politique, contactez-nous via l'interface de l'Application.
